
# コーディングルール

## 基本要素
- Common、Client、ServerはTypeScriptで開発する事
	- 各ファイルをtsフォルダの中に格納する事
- ファイルおよびクラスはなるべく分割する事
- クラス間の関係性は疎結合である事
- テストを書く事で契約的な実装である事を保証する
- すでにあるコードに倣うこと

## ライブラリの利用
- 不足するパッケージは追加してよい
